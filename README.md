# 👵 Amparo

Sistema de gestão de tarefas familiares voltado para o cuidado e acompanhamento de idosos.

---

## 📌 Sobre o Projeto

O **Amparo** é um software desenvolvido para auxiliar famílias na organização e no acompanhamento das atividades diárias, semanais e mensais de um idoso.

O sistema permite que membros da família registrem tarefas relacionadas ao cuidado, como:

- Administração de medicamentos  
- Consultas médicas  
- Atividades físicas  
- Rotinas alimentares  
- Exames periódicos  
- Compromissos diversos  

Cada família possui um administrador responsável pelo controle geral das atividades.

---

## 🎯 Objetivo

Facilitar a comunicação, organização e divisão de responsabilidades entre os membros da família, promovendo um cuidado mais estruturado, seguro e colaborativo.

---

## 👥 Perfis de Usuário

- **Administrador da família**
- **Membros da família**

---

## 🔐 Regras Gerais

- Todos os membros podem adicionar atividades.
- Apenas o administrador pode excluir atividades.
- As tarefas podem ser classificadas como:
  - Diárias
  - Semanais
  - Mensais
